# Weight Graph Feature Requests

## Feature Checklist

- [x] **Configurable moving average granularity** — Cog wheel icon to the left of the "+ Log Weight" button opens a settings panel directly below the graph. Users can adjust the moving average window (2–30 days) via a slider and see changes reflected in the chart in real time.

- [x] **Exclude current day's calories from the chart** — Today's calorie total is not plotted on the line chart (it is usually incomplete mid-day).

- [x] **"All" time range option** — An "All" chip has been added to the right of "5Y" in the time-range chip row.

- [x] **Year-only x-axis labels when timeline > 1.5 years** — When the visible date span exceeds ~548 days (1.5 years), the x-axis formatter switches to showing only the year (e.g. "2023") instead of "MMM d".

- [x] **Drag-to-zoom on the graph** — Users can drag horizontally on the chart; when they lift their finger the chart zooms in to exactly the range that was visible at the end of the drag gesture. Tapping any time-range chip resets the custom range.
